「社會價值國際研討會」首次在台舉辦 台灣表現受國際肯定

文 蔡淑芬 2019.12.04

 

「社會價值國際研討會」首次在台灣舉辦，由「社會價值國際」（Social Value International）與「台灣社會影響力研究院」（Social Value Taiwan）共同主辦，邀請產官學界共同與會，盼能將此概念推向主流社會。圖／台灣社會影響力研究院提供

 

英國「社會價值國際」組織（Social Value International）自2012年起為推動社會價值，每年選定一個國家舉辦一場國際研討會，足跡遍及德國、法國、米蘭、加拿大、英國、土耳其等國，2019年特別移師亞洲，並選定台灣為亞洲首站，今（4）日在台北華南銀行國際會議中心舉辦為期兩天「社會價值國際研討會2019」。金管會主委顧立雄受邀致詞時表示，金管會致力推動企業社會責任投資與鼓勵企業撰寫社會責任報告書，目的即在為提升社會價值做出貢獻。

 

「社會價值國際研討會2019」是由「社會價值國際」（Social Value International）與「台灣社會影響力研究院」（Social Value Taiwan）共同主辦，今年著重探討如何面對全球貧富不均與氣候變遷的挑戰。Social Value International CEO表示，人類正面臨許多挑戰，包括不平等及氣候變遷等，因此，希望透過倡導，喚起各界在進行決策時納入「社會價值」的觀念，並付諸實現。

 

吳嘉沅：倡議社會價值，創造更好人類未來

Social Value International在「社會價值國際研討會2019」會議前也特別頒發「全球社會價值」第一名給臺灣社會影響力研究院理事長吳嘉沅，以表揚吳理事長對提倡社會價值的努力與貢獻。吳嘉沅指出，他期望他的推動能減少世界上的不平等並對人類資源有更好的運用，創造更好的人類未來。

 

華南金控暨華南銀行董事長張雲鵬受邀致詞也表示，從管理層面思考，沒有辦法量化，就沒有辦法管理，社會價值也是如此，不過，國際上已發展多種方法衡量社會價值，透過量化管理可以讓投入資源者進行準確管理。華南集團也一直針對ESG投入不遺餘力，在企業社會責任上也受到肯定，例如受證交所的公司治理評鑑前5%企業，並納入台灣永續指數成份股，然而，華南也並沒有因此而自滿，除了減少碳排放，也不斷思考該怎麼做，以符合台灣所需要的社會價值。

 

立法委員余宛如則指出，我國在公司法修正時，第一條即提到企業除了追求利潤外，還加諸「追求社會目的與責任」，她認為，把追求社會價值的精神入法是也很有意義，也代表台灣追求向善社會的價值。

 

倡議社會價值走向主流 邀80位國內外講者分享

「社會價值國際研討會2019」邀請超過80位國內外講者，15場座談/論壇、6場工作坊、5場圓桌會議與6場學術場次，今年主題為「社會價值走向主流」，主辦單位邀請國內產官學貴賓蒞臨開幕式，如：金管會顧立雄主委與余宛如立委、華南金控張雲鵬董事長、資誠李宜樺會計師、國立中央大學周景揚校長及法鼓文理學院釋惠敏校長致詞。「社會價值國際」授予臺北市長柯文哲Global Champion Award，肯定將社會價值帶入公務系統。

 

五大主題演講邀請國際重量級講者，美國喬治城大學社會影響力與創新中心Lisa Hall、英國文化協會Tristan Ace、新加坡亞洲公益創投網絡Naina Batra、紐西蘭北島原住民委任局暨都會毛利人信託Awerangi Tamihere、及瑞士世界企業永續發展委員會Matthew Watkins。
